## Creating New Reports

1. Create a new directory: `docs/reports/<issue_number>/`
2. Write the report as `<issue_number>/<title>-report.md`
3. Update this index file to include the new report

## Report Template

```markdown
# Issue #<NUMBER> - <Title>

## 1. Problem Statement
## 2. Root Cause Analysis
## 3. Evidence from Codebase
## 4. Solution Proposals
## 5. Reference Examples
## 6. Recommended Solution
## 7. Implementation Checklist
## 8. Related Files
```
